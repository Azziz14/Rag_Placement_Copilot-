# Feature List

This document lists the core features of InterviewPilot-AI.

## Core Features

1. **Authentication**
   - Secure user sign-up, login, and session management.
   - Support for OAuth (e.g., Google, GitHub) and email/password authentication.

2. **Resume Parsing**
   - Extract candidate experience, skills, education, and achievements from uploaded PDFs/Word documents.
   - Structural and semantic analysis of resumes.

3. **Job Description Analysis**
   - Parse and analyze target job descriptions to extract key skills, requirements, and responsibilities.
   - Identify mismatch and match criteria against user resumes.

4. **Company-Specific RAG (Retrieval-Augmented Generation)**
   - Custom knowledge retrieval system tailored to specific target companies.
   - Integrates company values, style guides, and previous interview question banks.

5. **Mock Interviews**
   - Interactive, real-time simulated interviews (voice or text-based).
   - Dynamic questioning based on resume, job description, and company profile.

6. **Answer Evaluation**
   - Detail-oriented grading of responses using AI models.
   - Actionable feedback on communication style, technical accuracy, and domain alignment.

7. **Progress Tracking**
   - History of all mock interviews taken.
   - Topic/skill mastery tracking over time.

8. **Analytics Dashboard**
   - Rich visualizations of interview performance metrics.
   - Strengths and weakness breakdowns.
   - Recommended resources and personalized improvement paths.
